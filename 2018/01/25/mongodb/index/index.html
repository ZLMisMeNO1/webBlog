mongodb-索引

发表于 2018-01-25 | 分类于 nosql

数据准备

建立简单索引

db.collectionName.ensureIndex({key:1/-1})
1:正序索引 -1:反序索引

索引使用需要注意的地方

  1. 索引的创建在提高查询性能的同时会影响插入的性能。对于经常查询少插入的文档可以考虑用索引
  2. 使用索引注意索引的先后顺序
  3. 每个键建立索引不一定就能提高性能，索引不是万能的。
  4. 在做排序工作的时候，如果是超大数据量，也可以以考虑加上索引，用来提高排序的性能。
